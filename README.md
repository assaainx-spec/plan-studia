# Plan studiów

Plan zajęć na semestr IV psychologii klinicznej (2026 L).

Aplikacja webowa zoptymalizowana pod iPhone — działa w przeglądarce bez instalacji.

## Funkcje

- Podgląd nadchodzących zajęć pogrupowanych w zjazdy
- Dodawanie własnych zajęć i zadań (jednorazowych lub cyklicznych)
- Usuwanie i przywracanie zajęć z szablonu
- Dane zapisywane lokalnie w przeglądarce (localStorage)
- Ciemny motyw, obsługa safe area na iPhone (notch / home bar)

## Stack

- Next.js 16 · React 19 · TypeScript
- Tailwind CSS v4
